# Goblin-Hunter-3D

hello everyone this is my game project i made in Godot Engine. its a 3d action rpg where u have to save village from goblins. i work hard on the combat system so plz check it out.

About the Game:
u play as a warrior with sword. there is a Mage NPC who gives quest to kill goblins. if u kill them u get Gold and can buy potions (shop is working now).

Features i added:

    Sword combat with combo animations

    Ground Slam attack (jump and attack button)

    Blocking system (if u block at perfect time u take 0 damage!!)

    Quest system with dialogs

    Enemy AI that follows u and attacks

    Inventory and Gold system

CONTROLS:

Movement:
W, A, S, D = Walk
Spacebar = Jump
Shift = Dash (watch stamina!)
Mouse = Look around

Combat:
Left Click = Sword Attack
Right Click = Block (hold it)
Space + Left Click = GROUND SLAM (big damage)

Other:
E = Interact (Talk to NPC)
F = Torch light on/off
Esc = Pause game

plz play and tell me if bugs. thanks.
